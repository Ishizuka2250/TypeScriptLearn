# 現状スキル棚卸し（2026-05-18）

## 目的
2026年7月以降の「小規模在庫管理システム」案件に向けて、TypeScript / Next.js を1〜2か月で実務投入可能レベルに引き上げる。

## 学習目標（到達イメージ）
- TypeScriptで型安全なフロント・バックエンド実装ができる
- Next.js（App Router）でCRUD中心の業務アプリを構築できる
- 認証・DB接続・API設計・バリデーション・テストの基本を実装できる
- 在庫管理の業務要件を想定した小規模システムを自走で組み立てられる

## 既存スキル（入力情報の整理）

### 学生時代
- Cによる手続き型言語の基礎
- Javaによるオブジェクト指向の基礎
- DB基礎（MySQLクエリ作成、テーブル設計の基本）
- ネットワーク基礎
- Linux基礎（基本コマンド網羅）

### 実務
- PHP / Laravel 開発（フロント・バックエンド）: 約3年
- HTML / JavaScript 開発（フロント）: 約3年
- CSS: 基本理解あり（やや弱め）
- Excelマクロ（VBA）開発: 約6年

### 自主学習・趣味
- C# / VB.NET での Windowsアプリ開発
- Pythonでの簡易アプリ開発
- AWS基礎（SAA学習済、資格未取得）

## 強み
- Web開発の実務経験があるため、要件整理〜実装〜検証の流れを理解している
- Laravel経験により、MVC、ORM、認証、CRUDの勘所がある
- SQL・Linuxの基礎があり、バックエンド学習に移行しやすい

## 想定ギャップ
- TypeScript特有の型設計（Generics、Union/Intersection、型ガード）の実戦経験
- Next.js App Router（Server/Client Components、Route Handlers、Server Actions）の運用理解
- React設計（状態管理、フォーム、再利用可能コンポーネント設計）の体系化
- フロント品質（CSS設計・アクセシビリティ・UI整合）
- テスト実装（ユニット/結合/E2Eの最小セット）

- 毎週、動く成果物と振り返りを残す
- 学習時間の目安: 平日1.5〜2時間 + 休日3〜4時間
